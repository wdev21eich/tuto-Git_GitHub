# tuto-Git_GitHub

Installer [Git](https://git-scm.com/) 

Apprendre quelques commande Bash sur Git pour les sauvegardes en local et en ligne sur "GitHub".

Exemple de config :
```sh
$ git init
$ git config --global user.name "[nom]"
$ git config --global user.email "[adresse email]"
````
